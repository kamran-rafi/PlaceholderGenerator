# 🧱 Placeholder Image Generator

> Make Ugly Placeholders Beautifully Fast

A design-consistent placeholder image generator built for developers.  
Create images that actually match your UI — not random stock photos.

---

## ⚡ Why this exists

Most placeholder tools:
- Give you random images  
- Ignore your design system  
- Make your UI look off  

This tool fixes that.

**Bad placeholders ruin good design.**

---

## 🚀 Features

- 🎨 Custom background & accent colors  
- 📐 Set exact dimensions  
- 🧭 Optional dimension overlay (X & Y axis)  
- 😄 Add emoji (auto-scaled)  
- 🧩 Add SVG icons (Todo) 
- 📝 Add heading + description text  
- 💾 Export as:
  - PNG  
  - JPEG  
  - SVG  
- 📴 100% client-side (offline ready)  

---

## 🧠 Built for developers

- No API calls  
- No uploads  
- No tracking  
- Instant preview  
- Works entirely in your browser
